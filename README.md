# Philippine Military Academy (PMA) Awards System

A web-based awards management system for the Philippine Military Academy.

## Features

- User authentication (Login/Logout)
- Dashboard for overview
- Awards management sections:
  - Latin Honors
  - Saber Awards
  - Academic Awards
  - Plaque Awards
  - Streamer Awards

## Tech Stack

- **Frontend:** React.js
- **Routing:** React Router DOM
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Authentication:** JWT (JSON Web Tokens)
